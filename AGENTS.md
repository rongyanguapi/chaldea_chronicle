# AGENTS.md

## 项目通用规则

- 本项目是独立的纯 Flutter 工程，与 CamScanner 无业务、代码或平台耦合。参考外部项目规则或迁移 skill 时，必须移除 CamScanner 业务规则、内部平台、内部账号、内部接口、埋点平台等专属内容。
- 项目内新增或维护的 Codex skill 默认使用简体中文，包括 `SKILL.md` 正文、描述、提示语和面向开发者的说明。
- 新增代码注释、示例注释和文档注释默认使用简体中文。技术标识符、API 名称、命令、文件路径、Dart/Flutter 类型名和第三方库名称保持原文。
- 如果用户没有明确要求编译验证，不主动运行 Flutter/Gradle/Android/iOS 构建任务；优先通过静态检查、差异检查、搜索引用和必要的轻量级分析完成验证。
